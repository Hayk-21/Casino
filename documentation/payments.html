Payments for Crypto Casino

About

Payments is an add-on for the Crypto Casino application, which enables deposits and withdrawals via coinpayments.net API. This add-on can not be installed on its own without having Crypto Casino installed first.

System requirements

  • Crypto Casino 1.10.1 or higher.

Installation

Step 1

Log in to the Crypto Casino backend.

Step 2

Open Add-ons page, find the purchased add-on in the list and click Install button.

Step 3

Input the purchase code (it is sent by email from 2checkout.com) and click Proceed button.

The add-on should be automatically downloaded and installed.

Frequently Asked Questions

How to configure the Payments add-on?

Please sign up at coinpayments.net, obtain merchant ID, public and private keys and input them in the app settings in the backend.

Please also set the following permissions for the coinpayments.net API key (Account --> API keys --> Edit permissions):

  • get_basic_info
  • create_transaction
  • get_tx_info
  • balances
  • create_withdrawal
  • get_withdrawal_info
  • rates

For security reasons it's recommended that email confirmation is required for any withdrawals from your account (Account Settings --> Login & Security).

How is crypto deposit converted to credits?

In the app settings you can choose, which coin or fiat currency virtual credits are linked to (by default, USD) and its conversion rate (by default, 1 USD = 100 credits). This means that when a user deposits crypto (e.g. BTC, ETH etc) the app will first convert the received coins into USD using current market rate (provided by coinpayments.net) and then convert the result amount into credits using the above conversion rate.

How are withdrawals processed?

Withdrawals can be processed manually or automatically. If you want a want withdrawal request to be processed automatically you need to open the request by clicking "Edit" on Widthdrawals page in the backend and click "Approve" button. An automatic transaction to transfer the requested amount of coins from your CoinPayments.net wallet to the user wallet will be initiated. When this transaction is completed (upon receiving specific number of confirmations from the blockchain) the request will be marked as "Completed".

Otherwise if you want to process a withdrawal request manually you can send the required amount of coins directly from your wallet and then click "Complete" button to mark the request as "Completed".

Note that credits are automatically converted into coins according to the specified conversion ratio and current coin rates.

How to upgrade the add-on to a new version?

The application automatically checks for updates on a daily basis. If a new version is available a notice will be displayed on the Add-ons page in the backend. Simply click Update button to install the new version.

Please note that the game settings will be preserved, however any code changes will be overwritten.
